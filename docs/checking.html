Password Checking¶

class passwd.PasswordRequirements(*, min_length=0, min_digits=0, min_special=0, min_alpha=0, min_upper=0, min_lower=0, check_breaches=False, func=None)[source]¶

A set of requirements to check passwords against

Keyword Arguments

  • min_length (int) – The minimum length of the password

  • min_digits (int) – The minimum number of digits in the password

  • min_special (int) – The minimum number of special characters in the password

  • min_alpha (int) – The minimum number of alphabetical characters in the password

  • min_upper (int) – The minimum number of uppercase letters in the password

  • min_lower (int) – The minimum number of lowercase letters in the password

  • check_breaches (bool) – Whether to ensure that passwords aren’t found in known data breaches (uses check_password())

  • func (function) – A function that takes in a password (str) and returns a bool that must be True for the password to meet all requirements

passwd.check_password(password)[source]¶

Check a given password against known data breaches

Note

This method uses the Have I Been Pwned Passwords API. The unhashed password nor its full SHA-1 hash never leave the device.

Parameters
